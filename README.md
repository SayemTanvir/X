### Signal in the Silence.

**Local-Link HUD** is an offline-first, browser-based mesh communication tool designed for disaster relief, off-grid camping, and localized coordination without internet access.

It turns a single laptop into a **central command server**. Nearby users can connect via Wi-Fi/Hotspot to chat, share location data, and signal SOS‚Äî**no ISP or cellular data required.**

---

## üöÄ Key Features
* **100% Offline:** Works entirely on a Local Area Network (LAN) or Mobile Hotspot.
* **Real-Time Comms:** Instant chat powered by WebSockets (Socket.IO).
* **Emergency Radar UI:** A sci-fi inspired interface with "Panic Mode" SOS signaling.
* **Daisy-Chain Capable:** Extend range using Android "Wi-Fi Sharing" to relay signals between users.
* **Cross-Platform:** Works on any device with a browser (iOS, Android, Windows, Mac).
